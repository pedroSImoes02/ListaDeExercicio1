Verificação de Idade

if / else if

Par ou Ímpar

if / else + %

Digite um número inteiro e descubra se ele é par ou ímpar.

07

Situação do Aluno

else if

Digite uma nota de 0 a 10 e veja a situação: Aprovado (≥7), Recuperação (≥5 e <7) ou Reprovado (<5).

08

Positivo, Negativo ou Zero

ternário

Digite um número e o programa usará o operador ternário para dizer se é Positivo, Negativo ou Zero.
